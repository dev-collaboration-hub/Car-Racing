# Lightweight Car Racing Game 🏎️

A lightweight Car Racing game built from scratch with a focus on clean architecture, smooth controls, performance, and core game development fundamentals.

## 🎯 Project Goal

The goal of this project is to create a simple and efficient racing game implementation while understanding movement systems, physics, and real-time game mechanics.

## ✨ Features

* Car movement system
* Smooth driving controls
* Acceleration and braking
* Steering mechanics
* Track system
* Obstacle handling
* Collision detection
* Score tracking
* Game over handling

## 🏗️ Architecture

The project focuses on separating important game components:

* Game Loop
* Input System
* Rendering System
* Vehicle System
* Physics System
* Track System
* Collision System
* Score System
* Game State Management

## 🗺️ Roadmap

### M1 — Project Foundation

* Setup project structure
* Create game window
* Implement game loop
* Add FPS control

### M2 — Vehicle Mechanics

* Create car controller
* Add acceleration system
* Add braking system
* Implement steering logic
* Add movement physics

### M3 — Racing Systems

* Add track generation
* Implement obstacles
* Add collision detection
* Add scoring system
* Add difficulty progression

### M4 — Optimization & Release

* Improve performance
* Refactor architecture
* Add documentation

## 🛠️ Tech Stack

* Minimal dependencies

## 📌 Vision

Build a lightweight racing game that demonstrates real-time controls, physics simulation, and clean game architecture.
